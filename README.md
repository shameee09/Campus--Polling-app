# 📊 Campus Polling App

A real-time Campus Polling Application built using Dart and Firebase.
This app allows students to participate in polls, cast votes securely, and view instant results.

---

## 🚀 Features

* 🔐 User Authentication (Login / Register)
* 🗳️ Create and Manage Polls (Admin)
* 👤 Role-based Access (Admin / Voter)
* ✅ One User → One Vote
* 📊 Real-time Results Display
* 🔄 Automatic Role Redirection

---

## 🛠️ Tech Stack

* **Frontend:** Dart (Flutter)
* **Backend:** Firebase
* **Database:** Cloud Firestore
* **Authentication:** Firebase Auth

---

## 📂 Project Structure

* `main.dart` → App entry point
* `login_page.dart` → User login
* `register_page.dart` → User registration
* `auth_service.dart` → Authentication logic
* `admin_screen.dart` → Admin panel (create polls)
* `voter_screen.dart` → Voting interface
* `results_screen.dart` → Poll results
* `role_redirect.dart` → Role-based navigation
